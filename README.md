# manhuagui-dlr

manhuagui下載器  
純Python製作，執行過程不需要任何外部JS環境

包含自動webp轉檔jpg，可以選擇關閉

初始化完後就能完全放置，過程中只有主控台視窗，可以繼續使用電腦不受影響  

支援proxy與proxy pool，設定方式請參考更多資訊

執行完後可參考[這裡](https://github.com/HSSLC/kc-generator)做下一步處理成mobi檔案

## requirement:  
* lzstring
* requests
* Pillow
* BeautifulSoup4
